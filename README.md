# Data Analytics Virtual Experience Program (Forage)

## Overview

This repository contains my work from the Data Analytics Virtual Experience Program completed through Forage. The program focused on applying data analytics techniques to real-world business scenarios involving manufacturing telemetry data and workforce equality analysis.

---

## Task 1: Manufacturing Telemetry Analysis (Tableau)

In this task, I analyzed machine telemetry data collected from multiple factories across different countries.

### Objectives:

* Identify factory with highest machine downtime
* Analyze machine performance patterns
* Build interactive dashboards

### Work Done:

* Imported JSON data into Tableau
* Created calculated measures for downtime
* Built interactive dashboards
* Performed comparative analysis across factories
* Identified key operational issues

### Key Insight:

Certain factories experienced significantly higher downtime, indicating inefficiencies in machine operations and maintenance.

---

## Task 2: Workforce Equality Analysis (Excel)

In this task, I analyzed employee compensation data to evaluate fairness across job roles and factories.

### Objectives:

* Classify equality scores
* Identify fairness patterns in compensation

### Work Done:

* Used Excel for data transformation
* Created classification logic:

  * Fair (±10)
  * Unfair (outside ±10)
  * Highly Discriminative (extreme values)
* Analyzed workforce distribution

### Key Insight:

The analysis helped identify compensation imbalances across roles and factories.
